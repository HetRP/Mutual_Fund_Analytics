# Mutual Fund Analytics Platform

## Project Overview

This project analyzes mutual fund performance, investor behaviour, SIP trends, portfolio risk, and industry growth using Python, SQL, SQLite, and Power BI.

## Technologies Used

* Python
* Pandas
* NumPy
* SQLAlchemy
* SQLite
* Plotly
* Seaborn
* Power BI

## Project Structure

* data/
* notebooks/
* reports/
* dashboard/
* scripts/

## Setup Instructions

```bash
pip install -r requirements.txt
```

## Run ETL Pipeline

```bash
python run_pipeline.py
```

## Open Dashboard

Open:

dashboard/bluestock_mf_dashboard.pbix

using Power BI Desktop.

## Key Features

* ETL Pipeline
* SQLite Data Warehouse
* EDA Dashboard
* Risk Analytics
* VaR & CVaR
* Alpha/Beta Analysis
* Fund Recommendation Engine
